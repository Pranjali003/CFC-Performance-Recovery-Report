**4. Metric Cards Explained (Main Body)**
Each card follows the same structure, so once you explain one, the audience understands the rest.

**A. Inflammation ⚠️**
What is Inflammation?
Body’s response to stress, injury, or load
High inflammation = poor recovery

Data Displayed

Most Recent: Latest recorded value
14‑Days Avg / 30‑Days Avg: Short & medium-term trend
Bar graph: Historical pattern over time

Visual Signals

Yellow warning icon (⚠️) → Recovery risk flagged
Negative percentages → worse than the player’s baseline

What is Joint Range?

Range of motion and flexibility in joints
